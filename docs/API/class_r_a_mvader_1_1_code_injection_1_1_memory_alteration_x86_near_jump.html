Detailed Description

Represents a memory alteration that overwrites instructions of the target process' memory space with x86 NEAR JUMP instruction. ATTENTION: It is NOT recommended that you use NEAR JUMP instructions to jump to code caves that belong to automatically-allocated memory (which is the default behaviour of the Injector<TMemoryAlterationSetID, TCodeCave, TVariable> class). If you need to jump to a code cave in automatically-allocated memory spaces, you should strongly consider using MemoryAlterationX86FarJump instead of MemoryAlterationX86NearJump.
